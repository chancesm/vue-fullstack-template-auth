# VueJS App Skeleton with NodeJS Backend and Authentication

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Runs the backend services (express, db, etc);
```
npm run server
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
